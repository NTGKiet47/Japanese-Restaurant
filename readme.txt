2. Gặp lỗi khi import font awesome
3. Gặp lỗi background-image ko full phần tử: sử dụng backgroun-size: cover
4. Không sử dụng được carousel của bootstrap 
5. Chưa biết cách đọc dữ liệu từ file json để load lên giao diện, còn hỏi AI
